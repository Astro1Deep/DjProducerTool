# Registro de Cambios

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/es/1.0.0/),
y este proyecto sigue [Versionado Semántico](https://semver.org/spec/v1.0.0.html).

## [1.0.0] - 2024-01-04

### Agregado
- Suite de pruebas completa y corregida (`test_runner_fixed.sh`)
- Guías de contribución bilingües (CONTRIBUTING.md, CONTRIBUTING_ES.md)
- Validación completa de la estructura del proyecto
- Manejo mejorado de errores en scripts principales
- Mecanismos automáticos de recuperación de fallos
- Integración avanzada de ML con aislamiento local de venv
- Soporte para múltiples formatos de software DJ (Serato, Traktor, Rekordbox, Ableton)

### Corregido
- Lógica de validación de ruta de script corregida
- Navegación del directorio del test runner corregida
- Verificación de sintaxis mejorada para ambos scripts
- Problemas de permisos de archivo resueltos
- Mecanismos de backup mejorados para metadatos
- Algoritmos de detección de duplicados corregidos
- Proceso de recuperación de cuarentena mejorado

### Cambiado
- Estructura del proyecto reestructurada para mejor mantenibilidad
- Documentación actualizada para ambos idiomas
- Esquema de colores mejorado
- Sistema mejorado de navegación de menú
- Rendimiento optimizado para bibliotecas grandes

### Seguridad
- Protecciones mejoradas en modo seguro
- Verificación mejorada de integridad de backups
- Validación de suma de verificación agregada para archivos críticos
- Mecanismo de cuarentena mejorado con capacidades de restauración
